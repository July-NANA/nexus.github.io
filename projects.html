Project Card

Agentic AI Risk Traceability Audit System

Mar 2025 - Oct 2025
Problem

Large multi-agent systems create long chains of tool calls and interactions, which makes accountability, identity verification, and risk tracing difficult in regulated or high-stakes environments.

Data

The system processed agent interaction events, identity records, call traces, and workflow metadata generated by a large-scale multi-agent experimental platform supporting 100-level model interaction.

Approach

I combined a LangGraph-based multi-agent platform with backend service orchestration, GraphQL query interfaces, and blockchain smart contracts on Sui and Ethereum to register identities, record interaction events, and construct auditable trajectories.

Outcome/Impact

The project delivered an end-to-end audit and traceability mechanism for AI-agent interactions and improved traceability efficiency from O(n) to O(1), making verification and security review substantially faster.

Your Contribution

I worked as a researcher and development lead across backend service design, multi-agent platform construction, smart contract implementation, and traceability architecture.

Project Card

Huawei Network Equipment Operation Service

Jul 2021 - Jan 2023
Problem

Huawei needed faster and more scalable processing of large volumes of operational log data so engineering teams could detect faults, retrieve relevant events, and support network operations more efficiently.

Data

The project handled multi-source operational logs and real-time event streams at large scale, using log storage, preprocessing outputs, and indexed retrieval data to support millions of concurrent requests.

Approach

I built a real-time data processing pipeline with Kafka and Flink, optimized query schemes with MyBatis, MySQL, and ElasticSearch, and used S3-backed storage and backend services to improve ingestion, retrieval, and scalability.

Outcome/Impact

The system reduced log preprocessing latency from 5 minutes to 30 seconds and improved retrieval performance by 60%, strengthening fault detection efficiency and operational responsiveness.

Your Contribution

I contributed as a backend and big-data engineer, implementing core processing modules, query optimization, data storage integration, and cross-team technical delivery.

Project Card

Wafer Defect Monitoring Application

Jan 2023 - Aug 2023
Problem

Semiconductor manufacturing workflows required more efficient defect inspection and visualization so engineering teams could reduce manual screening effort and accelerate yield monitoring.

Data

The application used wafer inspection data, defect imagery, and processed image outputs stored across database records and object storage to support monitoring and visual review.

Approach

I developed backend services with Spring Boot and MySQL, connected them with S3-based storage, and supported image processing workflows using KNN, interpolation, and related methods to redraw and analyze wafer images.

Outcome/Impact

The project increased defect detection efficiency by 90% and reduced manual screening cost by improving visualization, query speed, and backend support for large inspection datasets.

Your Contribution

I was responsible for Java development, backend query optimization, database design, and image-algorithm-related implementation support.
